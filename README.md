# Gemini Realtime Chat Assistant

A Gemini-powered AI assistant built in Python with:

- Conversation memory
- Real-time search (with graceful fallback)
- Modular architecture
- Secure API key handling

## Features
- Gemini LLM integration
- Memory across conversations
- Optional real-time data
- Network-failure safe

## Setup
1. Clone repo
2. Create `.env`
3. Install dependencies
4. Run `python main.py`
